# Linear Regression using PyTorch nn.Module

## Overview

This notebook implements linear regression using PyTorch’s built-in modules such as `nn.Linear`, `MSELoss`, and optimizers.

## What This Covers

* Using `nn.Linear` for model definition
* Loss computation using `MSELoss`
* Optimizing parameters using `torch.optim`
* Training loop with forward and backward passes

## Key Concepts Learned

* Difference between manual implementation and PyTorch abstractions
* Role of optimizers in parameter updates
* Cleaner and scalable model building

## Workflow

1. Define model using `nn.Linear`
2. Compute predictions
3. Calculate loss
4. Backpropagate gradients
5. Update parameters using optimizer

## Result

The model learns to approximate a linear relationship between input and output using PyTorch utilities.
