# Tensor & Autograd Basics (PyTorch)

## Overview

This script demonstrates fundamental PyTorch concepts:

* Tensor creation and shapes
* Automatic differentiation (autograd)
* Gradient computation
* Basic tensor operations
* NumPy to PyTorch conversion

## Key Concepts Learned

* How gradients are computed using `.backward()`
* Importance of resetting gradients
* Tensor reshaping and operations

## Tech Used

* Python
* PyTorch
* NumPy
